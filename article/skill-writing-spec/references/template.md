# 最小模板

## SKILL.md 模板

```md
---
name: your-skill-name
description: 用一句话说明这个 skill 做什么，以及什么时候用。
---

# Skill Title

一句短说明。

## 必读引用

- [references/rules.md](references/rules.md)
- [references/structure.md](references/structure.md)
- [references/checklist.md](references/checklist.md)
```

## references 建议模板

```md
# 标题

## 目标

说明这个文件解决什么问题。

## 规则

1. 规则一
2. 规则二

## 反例（可选）

- 不推荐做法 A
```
