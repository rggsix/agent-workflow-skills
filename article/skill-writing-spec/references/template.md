# 最小模板

一句话定位：直接复制使用的最小骨架，避免从零组织结构。

## SKILL.md 模板

```md
---
name: your-skill-name
description: 用一句话说明这个 skill 做什么，以及什么时候用。
---

# Skill Title

一句短说明。

## 必读引用

- [references/rules.md](references/rules.md): 这个 skill 的硬规则。
- [references/checklist.md](references/checklist.md): 执行前后自检项。
```

## references 建议模板

```md
# 标题

一句话定位：说明这个文件只负责什么。

## 规则

1. 规则一
2. 规则二

## 边界（可选）

- 不做什么
```
