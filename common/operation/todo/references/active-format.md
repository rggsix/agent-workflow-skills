# active 格式

一句话定位：规定 `active.md` 的固定格式，保证简单可读。

## 固定模板

```md
# P0
- [ ] [需求标签] 待办事项简述 (YYYY-MM-DD)

# P1
- [ ] [需求标签] 待办事项简述 (YYYY-MM-DD)

# P2
- [ ] [需求标签] 待办事项简述 (YYYY-MM-DD)
```

## 格式约束

1. 任务必须包含创建日期 `(YYYY-MM-DD)`。
2. 不使用 `id`。
3. 不增加额外字段（如 `desc`/`see`/`status`）。
4. 描述必须一句话说清。如果关联file/class/function, 需要在描述中体现。

## 优先级

- `P0`: 阻塞问题或当前核心任务。
- `P1`: 重要改进，建议近期处理。
- `P2`: 可选优化，不影响主流程。
