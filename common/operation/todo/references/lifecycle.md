# 生命周期流程

一句话定位：定义新增、完成和归档流程，避免重复与历史污染。

## 新增流程

1. 新增前先快速扫描 `active.md`。
2. 若存在语义相同任务，更新原任务，不新增重复项。
3. 新任务按优先级追加到对应分区。

## 完成流程

1. 从 `active.md` 删除已完成任务。
2. 若 `archive/YYYY-MM.md` 不存在，先创建该月归档文件。
3. 追加到 `archive/YYYY-MM.md`。
4. 归档记录必须包含：
   - 原任务内容
   - 创建日期
   - 完成日期
   - 一句话完成原因

## 归档示例

```md
## 2026-02-18

- (P0) [需求标签] 待办事项简述
  创建: 2026-02-15
  完成: 2026-02-18
  原因: 一句话说明完成方式
```

## 禁止事项

- 不引入复杂字段。
- 不拆分为多个优先级文件。
- 不创建嵌套结构。
- 不修改历史归档文件。
- 不改变 `workspace` 规定路径。
