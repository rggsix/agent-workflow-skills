# 输出模板

一句话定位：统一规划输出，保证后续执行一致。

```md
## 规划产出

项目规范检查：...
项目记忆恢复：...
观察结果：...
约束要点：...

执行步骤（spec-lite）：
1. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...

2. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...

3. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...
```

# 不合格示例：
```
- 优化代码结构
- 调整状态逻辑
- 完善异常处理
```

# 合格示例：
```
项目记忆恢复：我将先读取 `project-memory` 的 `current.md` 与 `project-profile.md`，先恢复当前现场和长期稳定事实。
项目规范检查：我将检查是否有 project 相关 skills；已发现 `project` skill，将阅读并尝试在本环节应用其中与本次任务相关的规范。

1. 目标：`ChatSessionManager.swift` / `sendMessage()`
   动作：补充重复提交拦截
   约束：不改消息持久化结构
   完成标记：连续点击发送时仅触发一次请求
```
