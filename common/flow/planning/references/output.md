# 输出模板

一句话定位：统一规划输出，保证后续执行一致。

```md
## 规划产出

观察结果：...
约束要点：...

执行步骤（spec-lite）：
1. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...

2. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...

3. 目标：`文件 / 模块 / 方法`
   动作：...
   约束：...
   完成标记：...
```

# 不合格示例：
```
- 优化代码结构
- 调整状态逻辑
- 完善异常处理
```

# 合格示例：
```
1. 目标：`ChatSessionManager.swift` / `sendMessage()`
   动作：补充重复提交拦截
   约束：不改消息持久化结构
   完成标记：连续点击发送时仅触发一次请求
```
